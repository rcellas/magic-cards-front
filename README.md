# Prueba Angular

** Prueba de conocimientos en angular **


## Requirements
- Nodejs
- npm


## Installation Angular
### For windows
```
$ npm install -g @angular/cli
```

### For Mac or Linux
```
$ sudo npm install -g @angular/cli
```

## Installation proyect
```
$ npm install 
```

## Development server
```
$ ng serve
```
